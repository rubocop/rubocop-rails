* Fix false positives in `Rails/Pluck` when the receiver chain contains a method from the new `AllowedReceivers` config option. Defaults include Nokogiri traversal methods (`xpath`, `css`, `search`, `at_xpath`, `at_css`) so that `doc.xpath('//item').map { |node| node['name'] }` is no longer rewritten to a `pluck` call that the receiver does not support.
